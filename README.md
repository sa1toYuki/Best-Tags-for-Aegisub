# Best-Tags-for-Aegisub

Cansado de digitar tags manualmente? Bem, criei este script bem simples para sanar minha frustração disso, dá muito mais trabalho e atrasa os projetos, então resolvi criar este simples plug-in para você setar tags "automaticamente" no seu aegisub

Tired of entering tags manually?  Well, I created this very simple script to alleviate my frustration with this process, which takes a lot more work and delays projects, so I decided to create this simple plug-in for you to set tags “automatically” in your Aegisub. 

Instalação: 

Coloque o arquivo besttags.lua na pasta automation do seu aegisub e recarregue os scripts, ou feche e abra novamete e prontinho!

Instalation: 

Place the besttags.lua file in the automation folder of your Aegisub, then reload the scripts, or close and reopen the program, and you're all set!

Como usar: 

Na aba "automation" selecione "Best Tags" e digite sua tag de preferência sem os colchetes "{}"

How use: 

In the “Automation” tab, select “Best Tags” and enter your preferred tag without the square brackets “{}”.
